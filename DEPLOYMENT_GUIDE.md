# 🚀 菜谱大全APP - 快速部署指南

## 📋 项目概览

一个功能完整的安卓菜谱管理应用，包含：
- 🎲 随机菜品推荐功能
- 📚 完整的菜谱管理系统
- 💾 本地SQLite数据库存储
- 🎨 现代化Material Design界面

## 🛠️ 快速开始

### 1. 环境准备
确保您的开发环境已安装：
- Flutter SDK 3.0+
- Dart SDK 3.0+
- Android Studio 或 VS Code

### 2. 项目设置
```bash
# 进入项目目录
cd recipe_app

# 安装依赖
flutter pub get

# 检查环境
flutter doctor
```

### 3. 运行应用
```bash
# 启动开发版本
flutter run

# 构建发布版APK
flutter build apk --release
```

## 🔧 自定义配置

### 添加新菜谱
1. 点击"菜谱管理"Tab的"+"按钮
2. 填写菜谱信息：
   - 菜名和描述
   - 分类和难度
   - 烹饪时间
   - 食材清单
   - 制作步骤
3. 保存菜谱

### 修改分类
编辑 `lib/models/recipe.dart` 中的分类列表：
```dart
final List<String> _categories = [
  '川菜', '粤菜', '湘菜', '鲁菜', '苏菜',
  '浙菜', '闽菜', '徽菜', '家常菜', 
  '汤类', '凉菜', '甜品', '其他'
];
```

## 🐛 常见问题

### Q: 构建失败怎么办？
A: 运行 `flutter clean && flutter pub get` 清理重新构建

### Q: 数据库初始化失败？
A: 检查设备存储权限，确保有足够的存储空间

### Q: 图片不显示？
A: 检查图片路径是否正确，确保图片文件存在
