# 默认菜谱图片

这个文件夹包含菜谱APP使用的图片资源。

## 图片命名规范

- 格式：recipe_name.jpg
- 分辨率：建议 800x600 像素
- 格式：JPG 或 PNG

## 预置菜谱图片

- gongbao_chicken.jpg - 宫保鸡丁
- mapo_tofu.jpg - 麻婆豆腐  
- fish_flavored_pork.jpg - 鱼香肉丝
- white_cut_chicken.jpg - 白切鸡
- steamed_egg.jpg - 蒸蛋羹
- tomato_egg.jpg - 番茄炒蛋
- braised_ribs.jpg - 红烧排骨
- green_pepper_pork.jpg - 青椒肉丝
- sour_spicy_potato.jpg - 酸辣土豆丝
- steamed_bass.jpg - 清蒸鲈鱼
- default_recipe.jpg - 默认图片

## 添加图片

1. 将图片文件放入此目录
2. 确保文件名与数据库中的imagePath字段匹配
3. 建议使用有意义的文件名
