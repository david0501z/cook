## 🚀 部署说明

### 开发环境要求
- Flutter SDK 3.0+
- Dart SDK 3.0+
- Android Studio / VS Code

### 构建步骤
1. 运行 `flutter pub get` 安装依赖
2. 运行 `flutter run` 启动开发版本
3. 运行 `flutter build apk --release` 构建发布版本

### 安装测试
1. 启用设备"未知来源"应用安装
2. 传输APK文件到设备
3. 点击安装APK文件
4. 启动应用开始使用

## 📝 使用说明

### 随机推荐功能
1. 打开应用进入"随机推荐"Tab
2. 点击"换个推荐"获取新菜品
3. 点击菜谱卡片查看详细信息

### 菜谱管理功能
1. 切换到"菜谱管理"Tab
2. 使用搜索框查找特定菜谱
3. 点击分类标签筛选菜系
4. 点击"+"添加新菜谱
5. 长按菜谱进行编辑或删除

### 添加菜谱步骤
1. 点击右下角"+"按钮
2. 填写菜谱基本信息
3. 输入食材和制作步骤
4. 选择难度和分类
5. 点击"保存"完成添加
